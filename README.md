# Chat Websocket Java
This is a basic chat application with websockets that's programmed in java.

# Installation
* Open the Configuration.ini and change the connection data.
* Import the .sql file to your database
* Run the application

# Features
* Commands
* Prefixes
* Login
* Random information messages
* Badwords
* and more!
