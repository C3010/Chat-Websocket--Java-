# Chat Websocket Java
----

This is a basic chat application with websockets that's programmed in java.

### Version
----
0.0.1

### Installation
----

```sh
$ git clone [https://github.com/C3010/Chat-Websocket--Java-] chat
```

### Features
----

* Commands
* Prefixes
* Login
* Random information messages
* Badwords
* and more!
